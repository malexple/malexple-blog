## Правила тетраэдра в TetraGenesis

## Физика узлов

| Правило            | Описание                                                    | Параметр               |
| :----------------- | :---------------------------------------------------------- | :--------------------- |
| Пружина по ребру   | Каждое ребро — пружина Гука, тянет к равновесной длине `L0` | `k`, `L0 = 50`         |
| Отталкивание узлов | Несвязанные узлы отталкиваются если ближе `minDist`         | `minDist = 15`         |
| Тепловой шум       | Каждый тик к скорости добавляется случайный вектор          | `fluctAmp = 0.15`      |
| Демпфирование      | Скорость умножается на `damping` каждый тик                 | `damping = 0.9995`     |
| Интегрирование     | Явный Эйлер: `x += vx * dt`                                 | `dt = 0.5`             |
| Генератор шума     | Детерминированный, фиксированный                            | `rng = new Random(42)` |

## Непроницаемость стенок

| Правило                   | Описание                                                     | Параметр               |
| :------------------------ | :----------------------------------------------------------- | :--------------------- |
| Барьер грань→вершина      | Чужая вершина не проходит через треугольную грань            | `wallThickness = 6`    |
| Проверка проекции         | Барьер срабатывает только если проекция вершины попадает внутрь треугольника | —                      |
| Реакция при касании       | Квадратичная сила по нормали наружу                          | `wallK = 2.5`          |
| Реакция при проникновении | Дополнительное усиленное выталкивание                        | `wallK × 8`            |
| Импульс на грань          | Сила симметрично распределяется на три вершины грани         | share = `forceMag / 3` |
| Соседи исключены          | Собственные 4 вершины тетраэдра игнорируются при проверке    | —                      |

## Рост новых тетраэдров

| Правило            | Описание                                                     | Параметр                 |
| :----------------- | :----------------------------------------------------------- | :----------------------- |
| Триггер роста      | Только если натяжение рёбер тетраэдра превышает порог        | `tensionThreshold = 30`  |
| Грань роста        | Выбирается свободная грань (не общая с другим тетраэдром) с максимальным потоком скорости наружу | —                        |
| Апекс              | Стартует в центроиде грани, растёт по живой нормали каждый тик | `growthRate = 0.10 × L0` |
| Нормаль апекса     | Пересчитывается каждый тик по текущим A, B, C                | —                        |
| Знак нормали       | Фиксируется по противоположной вершине `D` исходного тетраэдра | —                        |
| Целевая высота     | Равновесная высота правильного тетраэдра                     | `√(2/3) × L0 ≈ 40.8`     |
| Запрет дубля роста | Нельзя запустить рост на грань с уже растущим апексом        | —                        |
| Запрет слипания    | Апекс не создаётся если слишком близко к любому существующему узлу | `minAllowed = L0/2`      |

## Топология

| Правило             | Описание                                                     |
| :------------------ | :----------------------------------------------------------- |
| Минимальная единица | Тетраэдр = 4 узла + 6 рёбер + 4 треугольные грани            |
| Общие вершины       | Соседние тетраэдры делят вершины — сдвиг одной вершины тянет все смежные грани |
| Свободная грань     | Грань, которую занимает ровно один тетраэдр — поверхность структуры |
| Внутренняя грань    | Грань между двумя тетраэдрами — барьер там не нужен          |
| Стенка непроницаема | Грань непроницаема для вершин **чужих** тетраэдров           |
